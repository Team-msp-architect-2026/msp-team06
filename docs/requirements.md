## 2-1. 서비스 한 줄 소개

구·동·아파트명 검색으로 실거래가·전월세 가격·주변 인프라·지역 뉴스를 한눈에 확인하고, AI 종합 리포트까지 제공하는 부동산 정보 지원 앱

| **외부 API**    | 도로명주소 API (행정안전부), 카카오맵 API                              |

| **외부 API**  | 국토부 아파트 매매 실거래가 API, 국토부 전/월세 실거래가 API                                                      |

## FR-07 AI 요약 리포트

# 9. 공통 UX 정책

| **항목**      | **정책**                                                    |
|---------------|-------------------------------------------------------------|

| 에러 처리     | API 오류 시 '데이터를 불러오지 못했습니다' 안내 메시지 표시 |

| 지원 지역     | 서울 전역. 타 지역 요청 시 422 UNSUPPORTED_REGION 반환      |

# 10. API 에러 코드

| **Status** | **errorCode**         | **상황**       | **설명**                                 |
|------------|-----------------------|----------------|------------------------------------------|
| 400        | INVALID_PARAMETER     | 파라미터 오류  | 요청 파라미터 누락 또는 형식 오류        |
| 401        | INVALID_API_KEY       | 인증 실패      | API Key 미제공 또는 유효하지 않음        |
| 404        | RESOURCE_NOT_FOUND    | 리소스 없음    | 요청한 지역/리소스 없음                  |
| 409        | REPORT_ALREADY_EXISTS | 리포트 중복    | 동일 캐시 조건 리포트 존재 (재사용 권장) |
| 422        | UNSUPPORTED_REGION    | 지원 범위 외   | 지원 범위 외 지역 (현재: 서울 전역)      |
| 429        | RATE_LIMIT_EXCEEDED   | 호출 한도 초과 | 분당 5회 / 일 20회 초과                  |
| 500        | INTERNAL_ERROR        | 서버 오류      | 서버 내부 오류                           |
| 503        | EXTERNAL_API_ERROR    | 외부 API 오류  | 외부 공공 API 연결 실패                  |
